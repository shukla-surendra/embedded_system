# Module 02 — Microcontrollers

A microcontroller is the brain of every embedded system — including your drone.

## Topics

## Goal

By the end of this module you will:
- Understand what a microcontroller is and how it differs from a computer's CPU
- Know the key specs to look at when choosing an MCU
- Understand why the ESP32 is a great choice for drone/IoT projects
