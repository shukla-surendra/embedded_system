# DIY Embedded Systems & Drone Electronics

A self-paced learning guide from zero to building your own drone — starting with the fundamentals of electricity and working up to ESP32 programming, sensors, and drone flight electronics.

---

## Learning Roadmap

```
Module 01 — Electronics Fundamentals
  └─ Voltage, current, resistance, components, circuits

Module 02 — Microcontrollers
  └─ What they are, how they differ from regular computers

Module 03 — ESP32 Basics
  └─ Board overview, setup, GPIO, blink LED

Module 04 — Communication Protocols
  └─ UART, I2C, SPI — how chips talk to each other

Module 05 — Sensors & Modules
  └─ IMU (MPU6050), GPS, ultrasonic, barometer

Module 06 — Power Systems
  └─ LiPo batteries, voltage regulators, ESCs

Module 07 — Drone Electronics
  └─ Motors, propellers, ESCs, flight controllers

Module 08 — Projects
  └─ Tying it all together — real builds
```

---

## How to Use This Repo

Each module is a folder with numbered markdown files. Read them in order. Each file ends with a **Practice** section — do those before moving on.

You don't need any hardware to read the theory modules (01 and 02). From Module 03 onward, you'll need an ESP32 dev board.

### Minimum Hardware for Getting Started
| Item | Purpose |
|------|---------|
| ESP32 development board (38-pin or 30-pin) | Main microcontroller |
| USB-A to Micro-USB cable | Programming & power |
| Breadboard (830 tie-points) | Prototyping circuits |
| Jumper wires (M-M, M-F, F-F assortment) | Connections |
| LEDs (assorted) + 220Ω resistors | Basic output |
| Multimeter (any cheap one works) | Measuring voltage/current |
